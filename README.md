Foi criado testes para todos tipos de operação em um crud (código criado em cima do divulgado na aula), incluindo: view, edit, delete, e create.
São criadas pré execução duas contas com valores diferentes e diferentes IDs, os testes consistem em verificar se os dados criados se mantém íntegros durante e no retorno do fetch, consultando nomes, saldos, exclusão. 
Portanto uma cobertura completa dos casos foi feita.
